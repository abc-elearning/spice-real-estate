### Long-term Vision
SpiceRE evolves from a single-user CLI tool into a comprehensive RE decision intelligence platform:

### v2 — Quantitative Enhancement
- Location Intelligence 8-tool toolkit (comparable, rental, infrastructure, legal, population, cycle, supply, cash flow verification)
- SQLite decision database replacing markdown archive
- Sensitivity analysis script
- Reflection loop with prediction tracking (compare predictions vs outcomes)
- User profile persistence

### v3 — Platform Expansion
- Claude Desktop / Claude.ai deployment
- Claude API integration
- Community-contributed personas and market sources
- Multi-region support (expanding beyond Vietnam)
- MCP server for external data feeds

### Strategic Priorities
1. **Validate with real decisions** — Use SpiceRE for actual investment decisions before adding features
2. **Cash flow accuracy** — Ensure OPEX calculations match Vietnam market reality through user feedback
3. **Bias detection quality** — Improve detection markers based on real conversation patterns
4. **Keep it simple** — Resist feature creep; one skill architecture until proven insufficient

### Moat
Consistency from structured cognitive loop > feature count. Ad-hoc prompting can't replicate the systematic bias detection + quantitative analysis + adversarial debate pipeline.
