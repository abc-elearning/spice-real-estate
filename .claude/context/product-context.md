# Product Context

### Target Users

1. **Cash Flow Investor (Primary):** Vietnamese RE investor prioritizing rental income over capital gains. Needs: net yield calculation with realistic OPEX, DSCR analysis, bias detection.
2. **First-time Buyer (Secondary):** Considering first purchase, needs buy vs rent comparison and FOMO detection.
3. **Existing Owner (Secondary):** Considering selling, needs loss aversion reframing and opportunity cost analysis.

### Core Use Cases

- "Should I buy this apartment for rental income?" → Full cognitive loop → Decision Card
- "Compare these 3 properties" → Multi-property analysis
- "Should I sell or hold this land?" → Prospect Theory reframe + bias detection
- "What if interest rates increase?" → Bayesian updating + stress test

### Key Value Propositions

- Bias detection catches psychological traps (anchoring, loss aversion, FOMO)
- Cash flow reality check exposes hidden OPEX (management, tax, maintenance, vacancy)
- DSCR < 1.0 flagged as RED FLAG before any recommendation
- 5-persona debate reveals hidden risks through structured disagreement
- Decision Card provides consistent, structured analysis output

### Disclaimer

SpiceRE is a decision SUPPORT tool, not financial advice. Every Decision Card includes mandatory disclaimer.
