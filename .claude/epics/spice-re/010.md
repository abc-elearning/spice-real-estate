# monte_carlo.py + utils.py — Price Projection & Shared Utilities

## Context

The simulation step of the cognitive loop needs probabilistic price projections. This script implements Geometric Brownian Motion for Monte Carlo simulation. Also creates the shared utils.py used by ALL scripts.

## Description

Create `scripts/utils.py` with shared JSON output, error handling, and input validation helpers. Create `scripts/monte_carlo.py` implementing GBM price simulation with numpy, CLI args, JSON stdout, fixed seed support, and stdlib fallback.

## Acceptance Criteria

- **FR-4 / Happy path:** `--price 3200000000 --growth 0.05 --std 0.08 --years 5 --sims 1000 --seed 42` → valid JSON with P10/P25/P50/P75/P90, prob_loss, mean
- **FR-4 / Deterministic:** Same seed → identical output on repeated runs
- **NFR-2 / Interface:** Accepts CLI args, outputs JSON to stdout, errors to stderr
- **NFR-3 / Reliability:** `--seed 42` produces identical results every run
- **FR-4 / Edge case:** `--price -100` → `{"status":"error","message":"price must be positive"}`
- **FR-4 / Fallback:** If numpy unavailable → falls back to stdlib random with stderr warning

## Implementation Steps

### Step 1: Create scripts/utils.py

- Path: `.claude/skills/spice-re/deciding-real-estate/scripts/utils.py`
- Functions:
  - `json_output(status="success", **data)` → prints JSON to stdout, includes all kwargs
  - `json_error(message)` → prints `{"status":"error","message":"..."}` to stdout, exits 1
  - `validate_positive(name, value)` → if value <= 0: json_error
  - `validate_range(name, value, min_val, max_val)` → if out of range: json_error
  - `validate_choice(name, value, choices)` → if not in choices: json_error

### Step 2: Create scripts/monte_carlo.py

- Path: `.claude/skills/spice-re/deciding-real-estate/scripts/monte_carlo.py`
- Argparse: --price (float, required), --growth (float, required), --std (float, required), --years (int, required), --sims (int, default 1000), --seed (int, optional)
- Validation: price>0, std>0, years 1-50, sims 100-100000
- Try import numpy; if ImportError → fallback to stdlib random with stderr warning
- GBM formula: S(t) = S(0) * exp((μ - σ²/2)*t + σ*W(t)) where W(t) = normal(0, sqrt(t))
- With numpy: vectorized simulation. With stdlib: loop-based.
- If seed provided: set numpy.random.seed(seed) or random.seed(seed)
- Output: compute percentiles P10/P25/P50/P75/P90 from final year prices, prob_loss = fraction where final < initial, mean
- json_output with all metrics
