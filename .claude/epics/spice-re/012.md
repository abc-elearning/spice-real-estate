# cashflow_proforma.py — 10-Year Cash Flow Pro-forma

## Context

This is the MOST IMPORTANT script for cash flow investors. It computes realistic net cash flow with all OPEX items that Vietnamese investors typically underestimate. DSCR < 1.0 detection is a critical safety feature.

## Description

Create `scripts/cashflow_proforma.py` building a 10-year monthly cash flow model with realistic OPEX, DSCR calculation and flagging, year-by-year breakdown. Conservative Vietnam-specific defaults.

## Acceptance Criteria

- **FR-3 / Positive CF:** Property with high rent/price ratio → positive monthly_cf, dscr_flag "green"
- **FR-3 / Negative CF:** Property where OPEX + debt > rent → negative monthly_cf, DSCR < 1.0, dscr_flag "red", warning message in output
- **FR-3 / DSCR ranges:** DSCR < 1.0 → "red", 1.0-1.3 → "yellow", ≥ 1.3 → "green"
- **NFR-2 / Interface:** CLI args → JSON stdout with summary + year_by_year array
- **FR-3 / Edge case:** rent=0 with loan>0 → DSCR=0, flag "red" (not division error)

## Implementation Steps

### Step 1: Create scripts/cashflow_proforma.py

- Import from utils: json_output, json_error, validate_positive, validate_range
- Argparse with all params:
  --price (float, required), --rent (float, required monthly gross rent)
  --vacancy (float, default 0.125 = 1.5 months/year)
  --mgmt_fee (float, default 10000 VND/m², per sqm/month — requires --area if provided)
  --area (float, optional sqm, for mgmt_fee calc)
  --tax_rate (float, default 0.10 = 10%)
  --maintenance (float, default 0.015 = 1.5% of price/year)
  --loan (float, default 0), --interest (float, default 0.09)
  --years (int, default 10)
  --rent_growth (float, default 0.03)
  --furniture (float, default 0 = total furniture cost, depreciated over 5 years)
- Validation: price>0, rent≥0, rates in [0,1], years 1-30
- For each year (1 to years):
  - gross_rent = monthly_rent * 12 * (1 + rent_growth)^(year-1)
  - vacancy_loss = gross_rent * vacancy
  - effective_gross = gross_rent - vacancy_loss
  - mgmt_cost = mgmt_fee * area * 12 (if area provided, else 0)
  - tax = effective_gross * tax_rate
  - maintenance_cost = price * maintenance
  - furniture_dep = furniture / 60 * 12 (if furniture > 0 and year <= 5)
  - total_opex = mgmt_cost + tax + maintenance_cost + furniture_dep
  - noi = effective_gross - total_opex
  - debt_service = loan * interest (annual, interest-only)
  - net_cf = noi - debt_service
  - Store in year_by_year array
- Summary metrics:
  - monthly_cf = year_1_net_cf / 12
  - annual_coc = year_1_net_cf / (price - loan) (equity)
  - dscr = noi_year1 / debt_service (if debt_service > 0, else null)
  - dscr_flag: "red" if dscr<1.0, "yellow" if 1.0-1.3, "green" if ≥1.3, null if no debt
  - If dscr < 1.0: add "warning": "⚠️ DSCR < 1.0 — negative cash flow. Monthly shortfall: X VND"
- Guard: if rent=0 and loan>0 → dscr=0, flag "red"
