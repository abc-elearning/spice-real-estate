### Step 2: Create BIASES.md
- Create `.claude/skills/spice-re/deciding-real-estate/BIASES.md`
- Port bias table from `docs/PRD_SpiceRE.md` §4.2
- For each of 12 biases, create entry with:
  - Name (English + Vietnamese)
  - RE-specific description
  - Vietnamese example scenario
  - Detection markers: 2-4 Vietnamese phrases/patterns (e.g., for anchoring: "tôi mua giá...", "giá gốc là...", "rao giá...")
  - Reframed question in Vietnamese (e.g., "Nếu bạn KHÔNG sở hữu BĐS này, bạn có mua ở giá hiện tại không?")
  - Severity for cash flow investors: High/Medium/Low
- Priority biases (anchoring, loss aversion, herding, disposition effect) get 4+ markers
- Other biases get 2-3 markers
