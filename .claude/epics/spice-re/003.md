## Context

The cognitive loop's Decision step needs persona definitions for adversarial debate, a Decision Card template for structured output, and Vietnam market data sources for web research. These are Level 3 reference files loaded on demand.

## Description

Create three reference files: PERSONAS.md defining 5 debate personas with conflicting viewpoints and tension relationships, DECISION-CARD.md as Vietnamese-language output template, and SOURCES-VN.md with Vietnam RE data sources and web search query patterns.

## Acceptance Criteria

- [ ] **FR-7 / Personas:** PERSONAS.md defines 5 personas: Veteran Investor, Cash Flow Investor, Risk Analyst, End User, Contrarian
- [ ] **FR-7 / Tensions:** Each persona has explicit tension relationships with ≥2 other personas
- [ ] **FR-7 / Key questions:** Cash Flow Investor persona includes DSCR, net yield, vacancy questions
- [ ] **FR-8 / Template:** DECISION-CARD.md has all required sections: situation, framework, bias alerts, scenario table, debate summary, recommendation with confidence %, next steps, disclaimer
- [ ] **FR-8 / Vietnamese:** Decision Card template text is in Vietnamese
- [ ] **NTH-1 / Sources:** SOURCES-VN.md has ≥10 data sources with query patterns for web_search

## Implementation Steps

### Step 1: Create PERSONAS.md
- Create `.claude/skills/spice-re/deciding-real-estate/PERSONAS.md`
- Port personas from `docs/PRD_SpiceRE.md` §4.4
- For each persona:
  - Name (Vietnamese + English)
  - Lens: what they focus on
  - Style: how they communicate
  - Tendency: their default bias/leaning
  - Key questions: 3-4 questions they always ask (Vietnamese)
  - Framework preference: which analytical framework they favor
  - Tension relationships: which personas they clash with and why
- The Cash Flow Investor persona must specifically ask about: DSCR, cash-on-cash, net yield, vacancy stress test, comparison with bank deposit
- Debate protocol: Present situation → each persona 3-5 sentences → identify tensions → synthesize

### Step 2: Create DECISION-CARD.md
- Create `.claude/skills/spice-re/deciding-real-estate/DECISION-CARD.md`
- Port template from `docs/PRD_SpiceRE.md` §4.1
- Template in Vietnamese with all sections:
  - Header: title, date, confidence level with percentage
  - Tình huống (Situation): 1-2 sentence summary
  - Phân tích theo framework: framework used + key results
  - Cảnh báo Bias: detected biases with impact and mitigation
  - Kịch bản (Scenarios): table with P10/P50/P90, probability, ROI, NPV
  - Tóm tắt Tranh luận (Debate Summary): table with persona, stance, key argument
  - Đề xuất (Recommendation): "Phân tích cho thấy..." framing + confidence %
  - Bước tiếp theo: 3 specific action items with timeline
  - Disclaimer: mandatory legal/advisory disclaimer in Vietnamese

### Step 3: Create SOURCES-VN.md
- Create `.claude/skills/spice-re/deciding-real-estate/SOURCES-VN.md`
- Port data sources table from `docs/PRD_SpiceRE.md` §4.5
- For each source: name, URL, data type, access method (web_search or web_fetch)
- Add web search query patterns for Claude's web_search tool:
  - Comparable: "giá [loại BĐS] [khu vực] [năm]"
  - Rental: "cho thuê [loại BĐS] [khu vực] giá"
  - Macro: "lãi suất cho vay BĐS [năm]"
  - Infrastructure: "quy hoạch [khu vực] [năm]"
- Include macro indicators interpretation table from §4.5
