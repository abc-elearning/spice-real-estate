# dcf_calculator.py — Discounted Cash Flow

## Context

Cash flow investors need NPV and IRR calculations to evaluate rental property investments against alternative investments.

## Description

Create `scripts/dcf_calculator.py` computing NPV, IRR (bisection method), cash-on-cash return, and equity multiple from rental income streams. Uses stdlib math only. Imports utils.py for shared patterns.

## Acceptance Criteria

- **FR-5 / Happy path:** `--monthly_rent 12000000 --growth 0.03 --opex_ratio 0.4 --discount 0.1 --years 10 --initial 3200000000` → valid JSON with npv, irr, cash_on_cash, equity_multiple
- **FR-5 / With loan:** Adding `--loan 1600000000 --rate 0.09` → adjusts cash flows for debt service
- **NFR-2 / Interface:** CLI args → JSON stdout
- **FR-5 / Edge case:** `--discount 0` → error JSON (division by zero prevention)
- **FR-5 / IRR no converge:** Cash flows that don't converge → `"irr": null, "irr_note": "IRR could not be determined"`

## Implementation Steps

### Step 1: Create scripts/dcf_calculator.py

- Import from utils: json_output, json_error, validate_positive, validate_range
- Argparse: --monthly_rent (float), --growth (float), --opex_ratio (float), --discount (float), --years (int), --initial (float), --loan (float, optional, default 0), --rate (float, optional, default 0)
- Validation: monthly_rent≥0, discount>0, years 1-50, initial>0, opex_ratio in [0,1]
- Build annual cash flows array: year_cf = (monthly_rent * 12 * (1+growth)^y) * (1 - opex_ratio) - annual_debt_service
- Annual debt service = loan * rate (interest-only simplified for v1)
- NPV = sum(cf_t / (1+discount)^t for t in 1..years) - equity (initial - loan)
- IRR: bisection method between -0.5 and 2.0, max 1000 iterations, tolerance 1e-6. If no convergence → irr: null with note
- Cash-on-cash = year_1_cf / equity
- Equity multiple = (sum all cf + terminal) / equity
